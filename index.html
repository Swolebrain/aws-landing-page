What is AWS?

AWS is changing the world. Basically, Amazon owns such a large cloud that they can lease any amount of virtual compute power or storage you need. They've also built hundreds of services that you can use as an AWS member, such as text message blasts, email blasts, media file conversions, networking in the cloud, encryption, and much more.

Introducing

AWS Solutions Architect Test Prep Course

Amazon Web Services (AWS) Certification is fast becoming a must-have for IT pros. This course is designed to help you pass the AWS Certified Solutions Architect (CSA) - Associate Exam, and introduces some of the topics present in the other two associate-level certifications (Developer and SysOps administrator). However, the course starts from the ground up, discussing what AWS is, the many services it offers, and how businesses are using the platform to find new ways to solve problems.


This course requires no prior knowledge of AWS and no programming knowledge. The course will be highly actionable in terms of showing how you can leverage AWS resources to accomplish common enterprise tasks such as speeding up a website by deploying CDNs, document versioning and backups, optimizing the cost of backups, creating virtual machines, and expanding your on-premise data center into a Virtual Private Cloud (VPC).
